### Installation

```sh
pip3 install brian_dash
```

or from source file


```sh
cd brian_dash # directory that have setup.py
pip3 install -e .
```


#### prerequisite:

```
dash-bootstrap-components
brian2
numpy
scipy
cython
dash
pandas
```
